## Ubunutu

* ddns 运行
```sh
curl https://github.com/NewFuture/DDNS/releases/download/v2.4.2/ddns -#SLo ddns && chmod +x ddns
```
* linux 创建定时任务(当前文件夹)
```sh
curl -sSL https://github.com/NewFuture/DDNS/releases/download/v2.4.2/create-task.sh | bash
```
